# 对齐原版 Lever-LM：全量候选池 + K=32

**启动：** 2026-06-05  
**目标：** train 全量 ~443,757 条建池，ICL 用 **K=32**（与当前一致），val **5000** unified 协议。

## 与原版对齐项

| 项 | 原版 | 本次 |
|----|------|------|
| 候选池 | ~44 万 train | **443,757 全 train** |
| 检索 K | top-K（论文子支持 64；ICL 常用 32） | **K=32** |
| Selector ckpt | 训于全库检索分布 | **先用现有 ckpt**（2 万池训的），有提升再 Step1 重生成 |

## 命令

```bash
# 后台全流程（建池 + ICL）
cd /home/jiyi/lizhiheng/Lever-LM
nohup bash scripts/run_fullpool_align.sh >> data/fullpool_align.log 2>&1 &

# 仅建池（断点续跑）
conda run -n leverlm python scripts/build_pool_only.py \
  --pool_out data/vqav2_pool_full \
  --pool_images_dir data/pool_images_full \
  --resume --checkpoint_every 500

# 查看进度
wc -l data/vqav2_pool_full.jsonl   # 已 embed 条数，目标 443757
tail -f data/fullpool_align.log
```
