## 四、恢复 GPU 后如何续跑

### 方式 A：一键续跑（推荐）

```bash
cd /home/jiyi/lizhiheng/Lever-LM
nohup bash scripts/run_base_full_pipeline.sh >> data/base_full_pipeline.log 2>&1 &
```

脚本逻辑（带 `--resume`）：
1. 跳过已有 `vqav2_pool_v2_base.json`
2. Step1 从 `vqav2_train_beams_v2_base.json` 已有条数继续（4400 → 5000）
3. Step1 完成后自动：切分 train/val → 训练 → ICL

### 方式 B：仅续 Step1

```bash
cd /home/jiyi/lizhiheng/Lever-LM
export PYTHONPATH=.
conda activate leverlm

python generate_data.py \
  --task vqa \
  --qwen_model /home/jiyi/.cache/modelscope/qwen/Qwen2-VL-2B \
  --anchor_random --pool_size 20000 --max_samples 5000 \
  --K 32 --beam_size 3 --shot_num 2 --score_batch_size 16 \
  --reward_mode info \
  --pool_in data/vqav2_pool_v2_base \
  --pool_out data/vqav2_pool_v2_base.json \
  --pool_images_dir data/pool_images_v2_base \
  --output_file data/vqav2_train_beams_v2_base.json \
  --save_every 200 --resume --seed 42
```
